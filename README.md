# Bonio Boost — Photo Booth Prototype

Interactive front-end prototype for the Bonio Boost photo booth (Liquid Glass UI).

- Portrait / landscape (tablet 3:4 · 4:3)
- Scenario controls: paid/free, payment success/fail, print success/fail
- Real idle-timeout screen

## Run locally
Just open `index.html` in a browser, or serve it:

    python3 -m http.server 8000
    # then open http://localhost:8000

## Live demo
Enable GitHub Pages (Settings → Pages → Deploy from branch → main / root).
